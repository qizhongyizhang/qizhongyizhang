1. 简介

1.1 发展历程

1978年由DanBricklin和BobFrankston创建世界上第一款电子表格——VisiCalc；

1983年，Lotus公司开发的一款成功的电子表格——Lotus 1-2-3；

1982年：微软推出第一款电子制表软件-Multiplan，败给了Lotus 1-2-3；

1985年：微软推出Excel for Mac；1987年：推出 Excel 2 for Windows；绑定系统，打败Lotus 1-2-3；随后收购Lotus公司

1993年：推出 Excel 5， Excel被捆绑进Microsoft Office中，Excel开始支持VBA；

1997年：推出 Excel 97，也称 Excel 8，Excel97是Office97中一个重要程序；通用叫法 97-03 版，文件后缀为 xls；03及以后文件后缀为 xlsx；xlsm（带宏的文件），本文操作以 97-03 版 为准

1.2 基本构成

工作簿：Excel文件又称为工作簿，由多个 sheet 组成；每个 sheet 包含功能区和操作区

功能区：由 文件、开始、插入、页面布局、公式、数据、审阅、视图 和 开发工具 等功能选项卡组成

操作区：由 1048576(行) * 16384(XFD，列) 的二维表格组成

2. 文件

2.6 选项

2.6.6 高级

自定义列表：点击 文件菜单栏 -> 选择高级 -> 向下滑动找到 编辑自定义列表(位置较下，需要滑动久一点)点击，调出 自定义序列 对话框 -> 对话框左侧选择新序列 -> 右侧输入需要加入的序列列表(序列值通过英文逗号 , 隔开，或者通过换行隔开) -> 点击添加 -> 点击确定，完成新下拉列表的添加；（修改：在 自定义序列 对话框左侧选择需要修改的序列 -> 右侧对序列值进行修改 -> 点击添加 -> 点击确定，默认序列不支持修改；删除：在 自定义序列 对话框左侧选择需要删除的序列 -> 点击删除 -> 点击确定）

3. 开始

3.1 剪贴板

3.1.1 粘贴-选择性粘贴

选择性粘贴打开方式：

  1. 点击 开始菜单栏 -> 点击 粘贴下拉框(最左侧) -> 选择 选择性粘贴，调出 选择性粘贴 对话框；
  2. 选择需要进行粘贴的单元格，点击鼠标右键，在出现的下拉列表中点击 选择性粘贴，调出 选择性粘贴 对话框

选择性粘贴-普通粘贴：点击 开始菜单栏 -> 点击 粘贴下拉框 -> 选择 选择性粘贴，调出 选择性粘贴 对话框 -> 选择 粘贴 中合适的粘贴方式；支持跳过空单元格进行粘贴

选择性粘贴-运算粘贴：点击 开始菜单栏 -> 点击 粘贴下拉框 -> 选择 选择性粘贴，调出 选择性粘贴 对话框 -> 选择 运算 中合适的粘贴方式；包括 加、减、乘、除（先复制进行运算的基数，再选择需要进行运算的数据，调出 选择性粘贴 的对话框，选择运算方式，点击确定即可得到结果）

3.7 编辑

3.7.2 填充

注：填充需要现在填充的开始单元格位置输入填充的起始值

  • 序列填充：选中填充的开始单元格位置 -> 点击 开始菜单栏 -> 点击 填充下拉选项卡，选择序列 -> 选择填充方向(行、列)、序列类型(等差、等比等)、填充步长和终止值

3.7.4 排序与筛选

  • 单列排序：选择进行排序的列 -> 点击 开始菜单栏 -> 点击 排序和筛选下拉框 -> 选择排序方式
  • 多列排序：点击 开始菜单栏 -> 点击 排序和筛选下拉框 -> 选择自定义排序 -> 在弹出的对话框中选择排序关键字和排序方式，可通过添加条件选择排序的次关键字和排序方式

3.7.5 查找和选择

  • 查找：点击 开始菜单栏 -> 点击 查找和替换(最右侧) -> 在下拉框中选择 查找 -> 在调出的查找对话框中点击选项，进行查找内容限定（内容、单元格格式）、范围限定（工作簿、工作表）、方式限定（按行、按列）等 -> 点击查找全部或查找下一个；快捷键操作：Ctrl + F -> 进行查找限定 -> 点击查找全部或查找下一个；默认会保留上一次查找限定
  • 替换：点击 开始菜单栏 -> 点击 查找和替换(最右侧) -> 在下拉框中选择 替换 -> 在调出的替换对话框中选项，进行替换内容限定（内容、单元格格式）、范围限定（工作簿、工作表）、方式限定（按行、按列）等 -> 在查找内容 和 替换为 的输入框中输入需要查找替换的内容 -> 点击全部替换或替换，可以提前选择需要替换的范围，则只会在选定范围内进行替换；快捷键操作：Ctrl + H -> 进行替换限定 -> 点击全部替换或替换；默认会保留上一次替换限定
    注：查找、替换支持通配符：? 代表1个字符，* 代表多个字符（包括 0 个）
  • 定位：先选择需要定位的范围，点击 开始菜单栏 -> 点击 查找和替换(最右侧) -> 在下拉框中选择 定位条件 -> 在调出的定位对话框中选择合适的定位条件 -> 点击确定，完成数据定位；此时可对所有选中的单元格进行操作(快捷键正常操作、完成输入操作后按 Ctrl + Enter 实现将内容填充至所有定位位置)；快捷键操作：Ctrl + G -> 选择定位区域 -> 选择定位条件 -> 点击确认

4. 插入

4.1 数据透视

数据透视表：更多操作查看 Excel进阶-数据透视表

4. 3 图表

图表：包括图表区、绘图区、图表标题、数据标签、数据系列、坐标轴、网格线、分类名称、图例、图表数据系列名称

  • 柱状图：在竖直方向比较不同类型的数据；适用于二维数据集，对于不同类型的数据进行对比，也可用于同一类型的数据在不同的时间维度的数据对比，通过柱子的高度来反应数据的差异
  • 条形图：水平方向比较不同类型的数据，更直观；类别名称很长，可以选择条形图
  • 折线图：按类别显示一段时间内数据的变化趋势，主要用于时间序列的数据；折线图适合二维的大数据集，还适合多个二维数据集的比较
  • 散点图：展示二维数据之间的相关关系；主要是展示两类数据之间的线性关系，前提是二者之间有很强的关联性
  • 气泡图：展示三维数据之间的关系；通过横纵坐标展示二维数据的关系，然后通过气泡大小来展示与第三个数据类型的关系
  • 饼图：也叫扇形图，主要描述的部分与整体的关系；描述一个整体的各个组成部分之间的占比情况
  • 雷达图：用于多维数据，多维度的综合对比；主要用于对比两种类型在多个维度的数据
  • 面积图：使用面积大小来展示数据对比情况；通过面积大小来对比不同类型的数据
  • 组合图：用多种不同的统计图形进行展示；当两种类型的数据在数值上差异很大的话，一种图表类型无法正常展示，可采用组合图

7. 数据

7.4 数据工具

  • 分列：选中需要分列的单元格，点击 数据菜单栏 -> 点击 分列(中部) ，调出 分列对话框 -> 在 分列对话框 中选择分列方式（特殊符号：分号、逗号、空格等，支持自定义符号；固定长度），点击 特殊符号 / 设置固定长度，点击下一步，支持在分列对话框下方进行预览 -> 选择数据输出格式（常规、文本、日期等）和输出位置，点击下一步 -> 点击确认，完成 将一个单元格内的数据拆分成多列
    拼接：=A1&”-“&B1 , 将A1、B1单元格的内容通过 - 拼接至公式输入的单元格

  • 删除重复值：点击 数据菜单栏 -> 点击 删除重复值(中部) ，调出 删除重复值对话框 -> 选择 重复值判定的列（即选定的列组合后重复，才判断为重复值；默认为全部列） -> 点击 删除重复项 -> 点击确定

12. 快捷键

12.1 菜单栏操作

收缩 / 展开菜单栏：双击即可实现

Ctrl + PgUp / PgDn：向右 / 左 切换 Sheet(工作表)

Ctrl + N：新建工作簿；Ctrl + W：不保存关闭当前工作簿

12.2 区域选择

home键：移动至当前行开头；Ctrl + home：回到 A1 单元格；Ctrl + Shift + home：选择当前单元格到 A1 单元格的矩形区域(要求是连续区域)

Ctrl + 上 / 下 / 左 / 右 箭头：向上 / 下 / 左 / 右定位至空白单元格；如果没有空白单元格，则连续定位至 当前列的 首行 / 末行 或者当前行的 首列 / 末列

Ctrl + Shift + 上 / 下 / 左 / 右 箭头：向上 / 下 / 左 / 右选择当前单元格至空白单元格的连续区域；如果没有空白单元格，则连续选择至 首行 / 末行 / 首列 / 末列 区域

12.3 F键相关

F2：在选中的单元格末尾进行编辑

F4：重复上次一次的操作；添加地址引用固定

F5：定位

F12：调出 另存为 对话框

12.4 函数

Alt + = ：求和

12. 其他

Ctrl + 1 ：调出 设置单元格格式 对话框

Ctrl + G：调出 定位 对话框

Ctrl + O：调出 打开 对话框，打开文件

Ctrl + P：调出 打印 对话框

Ctrl + R ：类似“复制+粘贴”功能，选择单元格和右方连续的需要填充的单元格，然后 Ctrl + R ，实现将单元格的内容填充至右方选中的所有单元格中（Ctrl + D：向下填充）

Ctrl + ; ：填充当前日期

Ctrl + 9 / 0：隐藏当前 行 / 列

Ctrl + ~ ：显示函数

Ctrl + M：合并单元格

Ctrl + E

13. 案例操作

案例1：将用科学计数法显示的过长数字显示出全部数字

  • 解决：通过分列实现
  • 操作：选择需要转换的列 -> 点击 数据菜单栏 -> 点击 分列(中部) ，调出 分列对话框 -> 在 分列对话框 中选择分列方式为 固定长度 -> 通过鼠标指定分列宽度（仅分一列） ，点击下一步-> 选择数据输出格式为 文本，点击下一步 -> 点击确认

案例2：拆分合并单元格并填充

  • 解决：全选拆分，公式填充，选择性粘贴
  • 操作：选择需要拆分的单元格 -> 拆分单元格(点击开始菜单栏，点击合并单元格) -> 定位空值(Ctrl + G，定位条件为空值) -> 公式填充(=A1(填充位置正上方的单元格)，Ctrl + Enter) -> 选择性粘贴为数值

案例3：标注两列存在差异的数据

  • 解决：定位（行内容差异单元格）、条件格式
  • 操作：
    1. 定位：先选中需要标注的内容 -> Ctrl + G -> 选择 行内容差异单元格，点击确定，完成定位 -> 添加合适的内容格式
    2. 条件格式：先选中需要标注的内容 -> 选择 开始选项卡 -> 点击 条件格式下拉框 -> 选择 突出显示单元格规则，选择 其他规则 -> 选择 使用公式确定要设置格式的单元格 -> 在单元格内部输入公式：=excat($A1, $B1) = False(判断A1、B1是否相等；仅返回 False-不相等的单元格) -> 点击格式，选择合适的标注格式 -> 点击确认
