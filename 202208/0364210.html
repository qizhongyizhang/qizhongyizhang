Pandas

Pandas：是一个开放源码、BSD 许可的库，提供高性能、易于使用的数据结构和数据分析工具；可以从各种文件格式比如 CSV、JSON、SQL、Microsoft Excel 导入数据，能够处理多种非数值型的数据，基于 NumPy 提供强大的分析结构化数据的工具集；Pandas 名字衍生自术语 “panel data”（面板数据）和 “Python data analysis”（Python 数据分析）；Github网址， Pandas官网

核心功能点：对各种数据进行运算，包括归并、再成形、选择，还有数据清洗和数据加工特征

  • Series：是一种类似于一维数组的对象，由一组数据（各种Numpy数据类型）以及一组与之相关的数据标签（即索引）组成；内部集成了处理字符串的多种函数
  • DataFrame：是一个表格型的数据结构，含有一组有序的列，每列可以是不同的值类型（数值、字符串、布尔型值）；DataFrame 既有行索引也有列索引，可以被看做 由 Series 组成的字典（共同用一个索引）
  • 支持多种数据类型，包括数组、标量、字符串、时间序列（时间偏移：Date offsets、重采样：Resampling）等
  • 多样的数据分析方法，包括索引：Index objects、分组聚合：GroupBy

其他功能点：

  • I/O编程：Input/output
  • 提供众多的通用函数：General functions
  • 窗口操作：Window
  • 其他：Style、Plotting、General utility functions、Extensions

1. Series

Series ：是一种类似于一维数组的对象，由一组数据（各种Numpy数据类型）以及一组与之相关的数据标签（即索引）组成；内部集成了处理字符串的多种函数

pandas.Series(data=None, index=None, dtype=None, name=None, copy=False, fastpath=False)

  1. data：Series 对象的数据；index：设置 Series 对象的数据索引；如果没有为数据指定索引，就会⾃动创建⼀个0到N-1（N为数据的⻓度）的整数型索引；
  2. dtype：数据类型；name：Series 对象的名称；
  3. copy：对象是否需要复制，默认为 False
import numpy as np
import pandas as pd

ser = pd.Series(data = [1, 2, 3, np.nan, 5, 6], index = ['a', 'b', 'c', 'd', 'e', 'f'], name = 'Series_Test')    # np.nan:表示空值

# 通过字典创建
student_dict = {'name':'xiaofang','age':30,'tel':1000}
ser_d = pd.Series(student_dict)

print(ser, '\n', ser_d)

2. DataFrame

DataFrame：是一个表格型的数据结构，含有一组有序的列，每列可以是不同的值类型（数值、字符串、布尔型值）；DataFrame 既有行索引也有列索引，可以被看做 由 Series 组成的字典（共同用一个索引）

pandas.DataFrame(data=None, index=None, columns=None, dtype=None, copy=None)

  1. data：DataFrame 对象的数据；index：设置 DataFrame 对象的数据索引；columns：列标签，默认为 RangeIndex (0, 1, 2, …, n)
  2. dtype：数据类型；copy：对象是否需要复制，默认为 False

2.3 排序

DataFrame.sort_index(axis=0, level=None, ascending=True, inplace=False, kind=’quicksort’, na_position=’last’, sort_remaining=True, ignore_index=False, key=None)

  1. axis：排序的方向；level：配置排序索引级别；ascending：配置是否升序排列，默认升序；inplace：执行后覆盖原数据
  2. kind：配置排序算法，有效值：’quicksort’, ‘mergesort’, ‘heapsort’, ‘stable’，默认为 ‘quicksort’；na_position：配置控制显示位置，有效值：’first’, ‘last’，默认为’first’
  3. ignore_index：配置排序后的索引，默认为 False，显示原索引，否则索引值为0,1,…,n-1

DataFrame.sort_values(by, axis=0, ascending=True, inplace=False, kind=’quicksort’, na_position=’last’, ignore_index=False, key=None)

  1. by：排序的列名；axis：排序的方向；ascending：配置是否升序排列，默认升序；inplace：执行后覆盖原数据
  2. kind：配置排序算法，有效值：’quicksort’, ‘mergesort’, ‘heapsort’, ‘stable’，默认为 ‘quicksort’；na_position：配置控制显示位置，有效值：’first’, ‘last’，默认为’first’；ignore_index：配置排序后的索引，默认为 False，显示原索引，否则索引值为0,1,…,n-1

def sa_str2nu(sa_str):
    if sa_str[-1] == "k":
        sa_num = int(sa_str[0:-1]) * 1000
    elif sa_str[-1] == "w":
        sa_num = int(sa_str[0:-1]) * 10000
    else:
        sa_num = int(sa_str)
    
    return sa_num

sa_avg = []
for i in sa_ser:
    sa_min = sa_str2nu(i[0])
    sa_max = sa_str2nu(i[1])

2.7 时间序列

pandas.date_range(start=None, end=None, periods=None, freq=None, tz=None, normalize=False, name=None, inclusive=None, **kwargs)

  1. start、end：时间序列开始、结束时间；periods：生成时间序列的个数；freq：时间序列的时间间隔，有效值：S(秒)、T(分，min)、H(时)、D(天)、W(周)、M(月)、Q(季度)、Y(年)，可以结合数字使用，默认为 D(1天)；tz：时区
  2. normalize：设置时间是否是午夜时间；name：时间序列的名称；inclusive：设置是否打开边界，有效值：’both’, ‘neither’, ‘left’, ‘right’，默认为：’both’
import pandas as pd

time = pd.date_range('2022/08/08 16:04', periods = 10, freq = '-10min')

time = time.tz_localize('UTC')            # 设置本地时区，tz = 'Asia/Shanghai'
time = time.tz_convert('Asia/Shanghai')   # 更改时区
print(time)

2.7.1 时间转化

pandas.to_datetime(arg, errors=’raise’, dayfirst=False, yearfirst=False, utc=None, format=None, exact=True, unit=None, infer_datetime_format=False, origin=’unix’, cache=True)，将字符串转化成时间日期类型

  1. arg：需要转换的时间；errors：对待转换错误的方式，有效值：’ignore’（忽略，返回传入的字符串）, ‘raise’（抛出一个错误）, ‘coerce’（将报错的数据设为 NaT），默认值：’raise’；dayfirst、yearfirst：是否日 / 年在前
  2. utc：转换的时区；format：转化的格式，具体见 时间格式化
import pandas as pd

file_path = './Data/pd_05_911.csv'

df_911 = pd.read_csv(file_path, encoding = "GBK")

df_911['timeStamp'] = pd.to_datetime(df_911['timeStamp'], format = "%Y-%m-%d %H:%M:%S")

2.7.2 重采样

重采样：指的是将时间序列从一个频率转化为另一个频率进行处理的过程，将高频率数据转化为低频率数据为降采样，低频率转化为高频率为升采样

DataFrame.resample(rule, axis=0, closed=None, label=None, convention=’start’, kind=None, loffset=None, base=None, on=None, level=None, origin=’start_day’, offset=None)

  1. rule：转换的规则；axis：转换的方向；closed：关闭的方向，有效值：’right’, ‘left’
# 统计每个月、每个季度的报警次数
import matplotlib

df_911.set_index('timeStamp', drop = False, inplace = True)
df_month_count = df_911.resample('M').count()[['title']]
df_quarter_count = df_911.resample('Q').count()[['title']]

plt.figure(figsize = (12, 7), dpi = 100)
plt.subplot(1, 2, 1)
plt.plot(df_month_count.index, df_month_count['title'])
plt.xlabel('Month')
plt.xticks(rotation = 45)
plt.grid()

plt.subplot(1, 2, 2)
plt.plot(df_quarter_count.index, df_quarter_count['title'], marker = '^')
plt.xlabel('Quarter')
plt.xticks(rotation = 45)
plt.grid()

plt.show()
