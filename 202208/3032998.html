写在前面：本文主要记录 数据分析 的相关基础知识

1. 简介

数据分析：用适当的统计分析方法对收集来的大量数据进行分析，运用高效的分析工具将他们加以分类和汇总，并提取其中最有价值的信息，概况总结形成有效结论，挖掘数据最大价值的过程，发挥数据的作用，支持企业决策

数据分析流程：

  1. 明确需求
  2. 确定思路
  3. 采集数据：八爪鱼、火车采集器、HAWK、Python(爬虫)
  4. 处理数据：MySQL、Kettle、Python(Numpy、Pandas)
  5. 分析数据：SPSS、Stata、Eviews、Python(机器学习)、R语言、MATLAB
  6. 展示数据：Tableau、PowerBI、FineBI、Python(matplotlib、Seaborn)、Echarts
  7. 撰写报告：Word、PPT
  8. 效果反馈

2. 数据分析方法

2.1 对比分析

将两个事物进行比较：横向比较（同一时间与它物进行对比）、纵向比较（不同时间自身对比）

同比：间隔时间的比较，如去年的今天和今年的今天进行比较

环比：相邻时间的比较，如本月和上个月进行比较

2.2 逻辑树分析

逻辑树：被称为问题树、演绎树或分解树，是麦肯锡公司提出的分析问题、解决问题的重要方法；将繁杂的数据工作细分为多个关系密切的部分，不断地分解问题，从而找出关键点，推动问题的解决

2.3 多维度拆解分析

从组成部分拆解

从指标构成拆解（用户：新老用户）

从业务流程拆解

2.4 归因分析（假设检验法）

  1. 提出假设（可以从产品 渠道 运营或4P营销理论进行大胆假设）
  2. 收集证据
  3. 得出结论

2.5 相关关系分析

相关关系分析：研究两种或两种以上的变量的关系

正相关：一个变量的增长会引起另外一个变量的增长；r = 1 代表完全正相关

负相关：一个变量的增长反而会引起另外一个变量的下降；r = -1 代表完全负相关

相关系数r的取值范围[-1,1]；|r|>0.6 代表相关性很强

2.6 RFM分析模型

  • 近度：Recency，最近一次消费到当前的时间间隔，指用户上一次消费的时间到目前统计时间的间隔
  • 频度：Frequency，最近一段时间内的消费次数，指用户在某段时间内的购物次数；消费频率越高意味着这部分用户对产品的满意度最高，用户粘性最好，忠诚度也最高
  • 额度：Monetory，最近一段时间内的消费金额，指用户在某段时间内的购物金额；消费金额较高的用户是需要重点争取的对象

通过RFM的取值可以将用户分成八大类

客户类型 R-客户黏性 F-忠诚度 M-消费力 对应场景 精细化运营
重要价值用户 高 高 高 优质客户，需要保持 VIP服务、个性化服务，附加销售
重要发展用户、重要深耕用户 高 低 高 优质客户，消费频率低；重点发展 交叉销售，提供会员忠诚计划，推荐其他产品
重要保持用户、重要唤回用户 低 高 高 优质客户，最近无消费；需唤回 DM营销，提供有用资源，续订产品或更新产品
重要挽留用户 低 低 高 优质客户，消费频率低，最近无消费；需挽留 重点联系，提高留存率
一般价值客户、潜力客户 高 高 低 低价值客户，消费频率高，最近有消费；需挖掘 销售高价值产品
一般发展客户、新客户 高 低 低 最近有交易的新客户；可推广留存 提供免费试用，提升客户兴趣，创建品牌和知名度
一般维持客户 低 高 低 价值贡献低，消费频率高 积分制，打折促销，保持联系
一般挽留客户、流失客户 低 低 低 可认为为流失客户 根据价值判断是否保留

2.7 漏斗分析模型

营销漏斗模型：指的是营销过程中，将非潜在客户逐步变为客户的转化量化模型；量化营销过程各个环节的效率，帮助找到薄弱环节

AARRR分析模型：

第一环节是获取用户（Acquisition）：关注渠道曝光量，渠道转换率，日新增用户数，日应用下载量，获客成本

  • 口碑渠道：适合病毒营销
  • 有机渠道：适合搜索引擎优化（SEO）、内容营销
  • 付费渠道：展示位广告、搜索广告、信息流广告

第二环节是激活用户（Activation）：关注日活，活跃率（活跃用户占比），PV，UV

第三个环节是提高留存（Retention）：关注留存率；重点关注留存率，复购率，人均购买次数，召回率等

第四个环节是增加收入（Revenue）：关注客单价、PUR付费用户占比、ARPPU某段时间内付费用户平均收入、生命周期价值（平均一个用户首次和最后登录之间创造的收入）、复购率、销售额（用户数转换率客单价*购买频率）

第五个环节推荐（Referral，病毒式营销）： 关注转发率、转化率、广告转化率、K因子（发出邀请人数*转换率，k>1表现好）

2.8 杜邦分析法

  1. 销售净利润：净利润/销售额

  2. 总资产周转率：销售额/总资产

  3. 权益乘数：总资产/净资产（自己出的钱）

业务问题分析：

  1. 明确问题：时间、地点、事件（比较对象、计算方式）
  2. 分析思路：多维度
  • 主体（人员、产品）
  • 时间
  • 空间
  • 环境（内部、外部：市场、竞争对手、重大事件）

常见分析指标

用户数据：

  • 日新增用户数：反映产品拉新的能力
  • 活跃用户数：衡量网站的运营现状，能够带来一些价值（打开应用、登录、产生特定的行为；分为 日活DAU：Daily Active User、月活MAU）；活跃率 = 活跃用户数 / 总用户数
  • 留存率：第一天访问的用户数在接下来指定天数的访问用户数的占比情况；反映产品留住用户的能力，行业基本标准（次日留存率：40%；7日留存率：20%；月留存率：10%）
  • 单位获客成本：广告活动产生的投放费用与广告活动带来的独立访客数的比值；单位访客成本与访客转化率、访客收入进行关联分析
  • 客单价：订单总额与订单数量的比值
  • ARPU：Average Revenue Per User，总收入/总用户数
  • ARPPU：Average Revenue Per Paying User，总收入/付费用户数

行为数据：

  • PV：页面访问量，也叫页面点击量，Page View；记录用户访问网页的次数，同一个网页刷新一次就算一次访问、用户对同一页面的多次访问，访问量累计
  • UV：独立访客数；通过cookie来标记用户，对同一cookie在一天内多次访问网站的用户仅记录为一个用户
  • IP：某IP地址的计算机访问网站的次数；如果是局域网使用同一个IP，则只记录一次
  • 平均访问时长：打开第一个页面到关闭最后一个页面的时长叫做总访问时长，平均访问时长 = 总访问时长 / 访问次数
  • 跳出率：跳出率 = 只访问一个页面就离开网站的访问次数 / 总访问次数；反映网站流量质量
  • 转化率：潜在用户在我们的网站上完成一次我们期望的行为叫一次转化；转化率 = 转化次数 / 访问次数；电商网站的转化率 = 网站下单的次数 / 网站总访问次数
  • 复购率：复购率 = 购买两次及以上的客户 / 总购买客户数

产品数据：

  • GMV：成交总额，流水；成交总额包括销售额、取消订单金额、拒收订单金额和退货订单金额
  • SKU：Stock Keeping Unit；库存量单位，SKU是商品信息聚合的最小单位
  • SPU：Standard Product Unit：标准化产品单元；SPU是商品信息聚合的最小单位，是一组可复用、易检索的标准化信息的集合，该集合描述了一个产品的特性
  • ROI：Return On Investment：投资回报率；产生的交易金额与活动投放成本金额的比值
