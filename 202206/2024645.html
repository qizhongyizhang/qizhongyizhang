写在前面：本文主要记录Python的面向对象编程基础知识

1. 简介

面向对象编程：Object Oriented Programming，简称OOP；把计算机程序视为一组对象的集合，一个对象包含数据和操作数据的函数，而每个对象都可以接收其他对象发过来的消息，并处理这些消息，计算机程序的执行就是一系列消息在各个对象之间传递；自定义的对象数据类型就是面向对象中的 类（Class）的概念；给对象发消息实际上就是调用对象对应的关联函数，我们称之为对象的 方法（Method）

面向对象最重要的概念就是类（Class）和实例（Instance）：
类 是抽象的模板，是一类具有相同特征和行为的事务，类拥有的属性（特征）和方法（行为）
实例 是根据类创建出来具体存在，每个对象都拥有相同的属性和方法，但各自的属性值可能不同

面向过程编程：Procedure Oriented Programming，简称POP；把计算机程序视为一系列的命令集合，即一组函数的顺序执行；为简化程序设计，面向过程把函数继续切分为子函数，即把大块函数通过切割成小块函数来降低系统的复杂度

2. 使用

2.1 自定义

# 处理学生的成绩表

        self.name = name                # Student类拥有name和score这两个类属性

2.1.1 封装

封装：将属性和方法封装到一个类中

2.1.1.1 实例属性与类属性

类属性：给类对象定义的属性，记录与类相关的属性，通过 c_name.attr_name 进行调用（建议），也可以通过 instance_name.attr_name 调用（向上查找机制，先找实例属性，在查找类属性）
实例属性：直接在class中通过 __init__ 方法定义的属性，类的所有实例都可以访问到，对具体对象实例申明的属性，仅本实例可以使用

class Student(object):
    student_count = 0               # 类属性，记录实例对象的数量
    def __init__(self, name):       # 初始化方法，用来定义实例属性
        self.name = name            # 实例属性，所有由Student类进行实例化创建的对象都有 name属性
        Student.student_count += 1  # 修改类属性的值

s = Student('Bob')
s.score = 90                     # 实例属性，仅Student类的实例化对象s有 score属性
# 实例属性优先级比类属性高，针对同一个属性，实例属性会屏蔽掉类属性
class Student(object):
    name = 'Student'

s = Student()
print(s.name)             # 输出 Student，可以通过 print(Student.name) 打印类的name属性

s.name = 'Student_s'
print(s.name)             # 输出 Student_s，实例属性屏蔽覆盖掉类属性

del s.name                # 删除实例的name属性
print(s.name)             # 输出Student，再次调用s.name，由于实例的name属性没有找到，就显示类的name属性

2.1.1.2 绑定方法

类方法：访问使用类属性，需要用 @classmethod 来标识，且第一个参数是 cls（习惯，表示类的引用），可以通过 cls. 访问类属性或调用类方法

实例方法：访问使用实例属性，第一个参数是 self

静态方法：既不需要访问实例属性，也不需要访问类属性的方法；需要使用 @staticmethod 来标识，不需要传递第一个参数（参数cls、self）

class Game(object):
    top_score = 0

    def __init__(self, player_name):
        self.player_name = player_name

    @classmethod                    # 定义类方法
    def show_top_score(cls):
        print("历史最高分: %d" % Game.top_score)

    @staticmethod                   # 定义静态方法
    def show_help():
        print('Show help.')

    def play_game(self):
        print('%s start play.' % self.player_name)

Game.show_help()
Game.show_top_score()
player_xm = Game('xm')
player_xm.play_game()

实例绑定

class Student(object):
    pass

def set_age(self, age):              # 定义一个函数作为实例方法
    self.age = age

s = Student()

from types import MethodType
s.set_age = MethodType(set_age, s)    # 前后都相同实例，给s实例绑定方法set_age,仅对当前实例有用

s.set_age(25)      # 调用实例方法
print(s.age)       # 测试结果

类绑定

def set_score(self, score):
    self.score = score

Student.set_score = set_score     # 给类绑定方法

s.set_score(100)
s2 = Student()
s2.set_score(99)                  # 所有Student类的实例都可以调用

print(s.score, s2.score)

**_slots**：限制类允许添加的属性；仅对当前类实例起作用，对继承的子类不起作用，如果在子类中也定义 __slots__，则子类实例允许定义的属性就是自身的 __slots_ 加上父类的 _slots_

class Student(object):
    __slots__ = ('name', 'age')  # 用tuple定义允许绑定的属性名称，只允许对Student实例添加name和age属性

2.1.1.3 私有属性和私有方法

私有属性：对象不希望公开的属性；私有方法：对象不希望公开的方法；通过 __ 表示，如：__name

class Women(object):
    def __init__(self, name, age):
        self.name = name
        self.__age = age

    def tell_age(self):         # 在类定义内部，可以使用私有方法
        print('%s的年龄是%d' % (self.name, self.__age))
    
    def __secret(self):
        print('%s的年龄是%d' % (self.name, self.__age))


women_xf = Women('xf', 18)
# print(women_xf.__age)         # 私有属性不能在类定义外部使用
women_xf.tell_age()
women_xf.__secret()             # 私有方法也不能在类定义外部使用
# 不要让内部属性不被外部访问，可以把属性的名称前加上两个下划线__，变成私有变量（private）
class Student(object):
    def __init__(self, name, score):
        self.__name = name          # 私有变量，不能在外部进行访问和修改
        self.__score = score        # 变量名类似__xxx__的，以双下划线开头，以双下划线结尾的变量是特殊变量，可以直接访问，不是private变量
    
    def get_name(self):             # 通过自定义get_name、get_score函数获取私有变量
        return self.__name

    def get_score(self):
        return self.__score

    def set_name(self, name):       # 通过自定义set_name、set_score函数修改私有变量
        self.__name = name

    def set_score(self, score):
        if 0 <= score <= 100:
            self.__score = score
        else:
            raise ValueError('error score!')

bart = Student('Bart Simpson', 59)
bart.__name = 'New Name'            # 并没有修改内部的 __name，而是给 bart对象新增了一个 __name变量；内部__name变量被Python解释器自动改成 _Student__name, 可通过该变量进行修改(不建议、不同解释器会改成不同的变量名)
print(bart.__name)       # 返回 New Name
print(bart.get_name())   # 返回 Bart Simpson

python中不存在真正的私有，python解释器针对私有属性和私有方法进行处理：在属性和方法名前增加 _类名，可以通过 _类名__属性名 或者 _类名__方法名 进行调用，不建议使用

print(women_xf._Women__age)
women_xf._Women__secret()

2.1.2 继承

继承：实现代码的复用，定义一个class时，可以从某个现有的class继承，新的class称为子类（Subclass）、派生类，被继承的class称为基类、父类或超类（Base class、Super class）；子类拥有父类所有的属性和方法，继承具有传递性：子类拥有父类所有的属性和方法，还拥有父类拥有父类所有的属性和方法

class Animal(object):
    def __init__(self, name):
        self.name = name
        # self.__name = name      # 无法继承父类的私有属性和私有方法
    
    def run(self):
        # print('%s is running.' % self.__name)
        return self.name + ' is running.'

class Cat(Animal):                # Dog类继承自Animal类，自动拥有 __name 属性和 run()方法
    pass

class Dog(Animal):
    def run(self):                # 重写父类方法，优先查找子类，后查找父类
        return 'Hello ' + self.name + '! Keep running.'

dog_ani = Animal('Wangwang')
cat_c = Cat('Mimi')
dog_d = Dog('Wangcai')

print(dog_ani.run())
print(cat_c.run())
print(dog_d.run())

多重继承：一个子类就可以同时获得多个父类的所有功能；在设计类的继承关系时，通常主线都通过单一继承下来；如果需要增加额外的功能，可以通过多重继承实现，通常称之为MixIn；MixIn的目的就是给一个类增加多个功能；优先考虑通过多重继承来组合多个MixIn的功能，而不是设计多层次的复杂的继承关系；多重继承时，如果多个父类含有重名的方法，则子类调用最左边父类的方法，可调用 son_c.__mro__ 内置类属性查看

class Land_Animals(object):
    def land_breathe(self):
        print('Breathe on land.')

class Aquatic_Animals(object):
    def water_breathe(self):
        print('Breathe in water.')

class Frog(Land_Animals, Aquatic_Animals):
    pass

frog_f = Frog()
frog_f.land_breathe()
frog_f.water_breathe()

2.1.2.1 继承方法重写

方法重写：父类的方法实现不能满足子类的需求，可以在子类中重新编写父类的方法实现，在子类中定义一个和父类同名的方法，并实现

super：是一个特殊的类，使用 super() 调用 super类创建的对象，super().fa_fun() 用来调用父类的方法；通常在子类重写父类方式时进行调用，实现父类的方法实现，同时在其他位置增加子类的特有的方法实现，也可以通过 fa_c.fa_fun(self) 在子类中调用父类的方法，在早期版本不支持 super

class Animal(object):
    def eat(self):
        print('Animal can eat.')

class Dog(Animal):
    def bark(self):
        print('Dog can bark.')

class XiaoTianQuan(Dog):
    def fly(self):
        print('XiaoTianQuan can fly')

    def bark(self):             # 对父类的bark进行重写
        print('I can speak.')

    def eat(self):
        super().eat()           # 通过super对父类方法进行扩展
        print('XiaoTianQuan can eat monster.')

xiaotian = XiaoTianQuan()
xiaotian.eat()
xiaotian.bark()
xiaotian.fly()

通过来说，子类不能继承父类的私有属性和私有方法；子类可以调用父类的共有属性和公有方法，父类的公有方法可以调用父类的私有属性和私有方法；通过在子类中调用父类的公有方法实现访问父类的私有属性和私有方法

2.1.3 多态

多态：不同子类对象调用相同的父类方法，产生不同的结果；可以增加代码的灵活度

2.1.4 获取对象信息

**dir(value)**：获得一个对象的所有属性和方法，返回一个包含字符串的list；可配合 getattr()、setattr()、hasattr()，直接操作一个对象

class MyObject(object):
    def __init__(self):
        self.x = 9
    
    def power(self):
        return self.x * self.x

obj = MyObject()
print(hasattr(obj, 'x'))  # 判断 obj对象 是否有 x 属性
setattr(obj, 'y', 19)     # 新增一个属性'y'，值为 19
print(dir(obj))           # 返回 obj对象 的所有属性和方法
print(getattr(obj, 'y'))  # 获取不存在的属性，会抛出AttributeError的错误; 可以设置默认值:getattr(obj, 'z', 404):获取属性'z'，如果不存在，返回默认值404; fn = getattr(obj, 'power'):获取属性'power'并赋值到变量fn

2.1.5 类内置属性和方法

  • **__new__**：方法，创建对象时，自动调用；为对象分配内存空间
  • **__init__**：方法，对象初始化时，自动调用；用来定义一个类具有的属性
  • **__del__**：方法，对象从内存中销毁时，自动调用

2.1.6 @property

@property：把一个方法变成属性调用；把一个getter方法变成属性，只需要加上 @property 就可以；此时，@property 又创建了一个装饰器 xxx.setter ，负责把setter方法变成属性赋值

class Student(object):
    @property
    def score(self):
        return self._score

    @score.setter
    def score(self, value):
        if not isinstance(value, int):
            raise ValueError('score must be an integer!')
        if value < 0 or value > 100:
            raise ValueError('score must between 0 ~ 100!')
        self._score = value

s = Student()
s.score = 60            # OK，实际转化为s.set_score(60)
print(s.score)          # OK，实际转化为s.get_score()

# s.score = 9999        # 报ValueError

2.2 定制类

定制类：通过有特殊用途的函数，帮助我们完成类的定制，实现某些特定功能

  • **__new__()**：为对象分配内存空间；重写 __new__ 方法一定要返回 **super().__new__(cls)**，__new__ 是一个静态方法，在调用时一定要传递 cls 参数
class MusicPlayer(object):
    def __new__(cls, *args, **kwargs):
        instance = super().__new__(cls)
        return instance

    def __init__(self):
        print('Initialization.')

wangyi = MusicPlayer()
print(wangyi)

# 单例类：只生成一个实例，即对同一个类的不同实例对象只返回一个内存地址
class MusicPlayer(object):
    instance = None      # 记录第一个对象的引用

    def __new__(cls, *args, **kwargs):
        if MusicPlayer.instance is None:
            MusicPlayer.instance = super().__new__(cls)
        return MusicPlayer.instance

    def __init__(self):
        print('Initialization.')

wangyi = MusicPlayer()
qq = MusicPlayer()
print(wangyi)
print(qq)
  • **__str__()：定义类实例打印模块，返回对象的描述信息（必须返回字符串），与 print函数配合使用，面向用户；__repr__()**：面向程序开发者，二者代码都是一样的
class Student(object):
    def __init__(self, name):
        self.name = name
    
    def __str__(self):
        return 'Student object (name: %s)' % self.name     # 必须返回字符串
    
    __repr__ = __str__        # 简单写法，也可重新定义一个 __repr__

print(Student('Michael'))     # 输出 Student object (name: Michael)；如果不定义__str__()，则print会输出 <__main__.Student object at 0x000001F6EC352370>：表示有一个Student类，且Student类的内存地址为0x000001F6EC352370

  • **__iter__()**：返回一个可迭代对象类，被用于 for … in 循环；调用 __next__() 方法拿到循环的下一个值

    class Fib(object):
    def __init__(self):
        self.a, self.b = 0, 1                       # 初始化两个计数器a，b

    def __iter__(self):
        return self                                 # 实例本身就是迭代对象，故返回自己

    def __next__(self):
        self.a, self.b = self.b, self.a + self.b    # 计算下一个值
        if self.a > 100000:                         # 退出循环的条件
            raise StopIteration()
        return self.a                               # 返回下一个值

for n in Fib():
    print(n)

  • **__getitem__()**：Fib实例虽然能作用于for循环，但还是不能把它当成list来使用，无法直接获取某个元素，也无法进行切片操作

class Fib(object):
    def __getitem__(self, n):
        if isinstance(n, int):      # n是索引，获取指定值
            a, b = 1, 1
            for x in range(n):
                a, b = b, a + b
            return a
        if isinstance(n, slice):    # n是切片，进行切片
            start = n.start
            stop = n.stop
            if start is None:
                start = 0
            a, b = 1, 1
            L = []
            for x in range(stop):
                if x >= start:
                    L.append(a)
                a, b = b, a + b
            return L

f = Fib()
print(f[2])
print(f[0:5])
# 仍然存在缺陷，没有对step参数和负数作处理；对应的还有__setitem__()方法：把对象视作list或dict来对集合赋值；__delitem__()方法：删除某个元素
  • **__getattr__()**：返回动态类的一个属性，属性不存在于 __init__() 时调用
class Student(object):
    def __init__(self):
        self.name = 'Michael'

    def __getattr__(self, attr):
        if attr=='score':
            return 99
        if attr=='age':               # 也可以返回函数，调用方式要变为函数调用
            return lambda: 25
        raise AttributeError('\'Student\' object has no attribute \'%s\'' % attr)    # 只对定义的属性响应，定义外的属性仍然抛出 AttributeError；未定义就返回 __getattr__() 的默认返回值 None

s = Student()
print(s.name)
print(s.score)
print(s.age())
print(s.abc)
  • **__call__()**：通常调用实例方法时，通过 instance.method() 来调用；可以通过定义一个 __call__()方法，直接对实例进行调用
class Student(object):
    def __init__(self, name):
        self.name = name

    def __call__(self):      # 还可以定义参数。直接调用实例进行与函数调用一样，传入参数
        print('My name is %s.' % self.name)

s = Student('Michael')
s()            # self参数不需要传入

2.3 枚举类

枚举类：存储枚举的常量的类，有特殊用途的函数，可以帮助我们定制类

from enum import Enum

Month = Enum('Month', ('Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec'))    # 创建Month类型的枚举类

print(Month)                           # 打印实例
print(Month.__members__)               # 打印元素
print(Month.__members__.items())       # 打印元素，字典形式
print(Month.__members__.keys())        # 打印属性，字典形式
print(Month.__members__.values())      # 打印属性值，字典形式；value属性则是自动赋给成员的int常量，默认从1开始计数

for name, member in Month.__members__.items():
    print(name, '=>', member, ',', member.value)
from enum import Enum, unique

@unique                 # @unique装饰器可以保证没有重复值
class Weekday(Enum):
    Sun = 0             # Sun的value被设定为0
    Mon = 1
    Tue = 2
    Wed = 3
    Thu = 4
    Fri = 5
    Sat = 6

print(Weekday.Tue)            # 根据属性取值，返回 Weekday.Tue
print(Weekday['Tue'])         # 根据key取值，返回 Weekday.Tue
print(Weekday.Tue.value)      # 获取属性对应值，返回 2
print(Weekday(1))             # 根据值获取属性，返回 Weekday.Mon；Weekday(7)，值不在定义内，会报 ValueError

day1 = Weekday.Mon
print(day1)                   # 可以给其他变量赋值，返回 Weekday.Mon
print(day1 == Weekday(1))     # 可以进行比较，返回 True
