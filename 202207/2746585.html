写在前面：本文主要记录Python模块-matplotlib-的基础知识

matplotlib

matplotlib：图形绘图库，可以用来绘制各种静态，动态，交互式的图表，包括线图、散点图、等高线图、条形图、柱状图、3D 图形、图形动画等；由 John D. Hunter 编写，大部分使用 python 编写（为平台兼容，少部分使用 C 、 Objective-C 和 JavaScript）；Github网址， matplotlib官网， matplotlib官网样例

安装：pip install matploylib，可通过 print(matplotlib.__version__) 验证，正常输出版本号则正常安装
导入：import matplotlib

1. 画布操作

1.1 绘制画布

绘制画布：matplotlib.pyplot.figure(num=None, figsize=None, dpi=None, facecolor=None, edgecolor=None, frameon=True, FigureClass=<class ‘matplotlib.figure.Figure’>, clear=False, **kwargs)

  1. num：设置画布的唯一标识符；figsize：设置画布大小，默认为 (6.4, 4.8)，单位是英寸；dpi：设置画布的分辨率(网络每英寸点数)，默认为100
  2. facecolor：设置背景颜色(线框外侧)；edgecolor：设置边缘颜色；frameon：设置事后绘制外边框
import matplotlib.pyplot as plt
fig = plt.figure(num = 1, figsize = (12, 7), dpi = 100, facecolor = 'r', edgecolor = 'k', frameon = False)

xpoints = [1, 2, 3, 4, 5, 6]
ypoints = [12, 14, 15, 17, 20, 25]

plt.plot(xpoints, ypoints)
plt.savefig("./matplotlib_fihure.png")     # 保存画布，需要在 show() 方法之前

plt.show()

2. 绘图

PyPlot：最常用子模块 import matplotlib.pyplot as plt

绘图：matplotlib.pyplot.plot([x], y, [fmt], *, data=None, **kwargs)，绘制折线图

  1. [x], y：传入两个参数，分别指定x、y轴的坐标，要求坐标数据数量必须一致；如果省略x轴数值，则默认为0,1,2,3… 自动匹配
  2. fmt：通过字符串指定 颜色字符、线型字符、标记字符；不区分顺序
  3. **kwargs：通过关键字方式指定 Line2D properties属性，包括 color：颜色字符、linestyle：线型字符、linewidth：线性宽度、marker：标记字符、markersize：标记大小
# 1. [x], y
import matplotlib.pyplot as plt
fig = plt.figure(figsize = (12, 7), dpi = 100)

xpoints = [1, 2, 3, 4, 5, 6]
ypoints = [12, 14, 15, 17, 20, 25]

plt.plot(xpoints, ypoints)
# plt.plot(ypoints)
plt.show()

import matplotlib.pyplot as plt
fig = plt.figure(figsize = (12, 7), dpi = 100)

xpoints = [1, 2, 3, 4, 5, 6]

plt.plot(xpoints, ypoints_2, 'b--s')
plt.show()

2.1 设置标签

设置标题标签：matplotlib.pyplot.title(label, fontdict=None, loc=None, pad=None, *, y=None, **kwargs)
设置 x轴 标签：matplotlib.pyplot.xlabel(xlabel, fontdict=None, labelpad=None, *, loc=None, **kwargs)
设置 y轴 标签：matplotlib.pyplot.ylabel(ylabel, fontdict=None, labelpad=None, *, loc=None, **kwargs)

  1. label、xlabel、ylabel 设置标题、x轴、y轴 标签名称，如果是中文需要设置中文显示字符
  2. fontdict：设置标签字体属性字典，可以和 **kwargs 结合使用
  3. loc：设置标签位置，title、xlabel有效值：’left’, ‘center’, ‘right’，默认为居中；ylabel有效值：’bottom’, ‘center’, ‘top’，默认为居中
  4. pad、labelpad：设置标签和坐标轴的距离，默认为4
  5. y：设置标题的纵轴位置，接收float数据，默认为1，在图表顶端，0.5则位于图表中部
  6. **kwargs：通过关键字方式指定 text properties属性，指定 fontstyle：字体风格，简写为 style，有效值：’normal’, ‘italic’(斜体), ‘oblique’、fontsize：字体大小，或者size、color：字体颜色，简写为 c、rotation：旋转角度，有效值：’vertical’(0), ‘horizontal’(90)，可以输入数值
import matplotlib.pyplot as plt

# 设置显示中文字体：黑体
# 设置字体字典
# font = {'family': 'MicroSoft YaHei',
#        'weight': 'bold',
#        'size': 'larger'}

# matplotlib.rc('font', font)

# 打印所有系统字体
# fc-list  ，fc-list :lang=zh
# print([f.name for f in matplotlib.font_manager.fontManager.ttflist])

from pylab import mpl
mpl.rcParams["font.sans-serif"] = ["SimHei"]

# 自定义字体
# 实例化字体对象：fname：字体库访问路径，size：设置字体大小
myfont = matplotlib.font_manager.FontProperties(fname="C:\Windows\Fonts\Candara.ttf", size = 24)

fig = plt.figure(figsize = (12, 7), dpi = 100)

xpoints = [1, 2, 3, 4, 5, 6]
ypoints = [12, 14, 15, 17, 20, 25]

plt.plot(xpoints, ypoints)

# 定义标签字体字典
font_dict=dict(fontsize = 16, color = 'b', style = 'oblique')

# 定义图表标题标签
plt.title("matplotlib graph", fontproperties = myfont, y = 1.1)
# plt.title("matplotlib graph", fontdict = myfont, y = 1.1)      # 有报错，但能显示

# 定义坐标轴标签
plt.xlabel("月份", font_dict, labelpad = 10, loc = "right")
plt.ylabel("数量", font_dict, loc = "top", rotation = 'horizontal')

plt.show()

2.2 设置坐标轴刻度

设置坐标轴刻度：matplotlib 会自动生成对应坐标轴的刻度，也可以自定义

matplotlib.pyplot.xticks(ticks=None, labels=None, **kwargs)

matplotlib.pyplot.yticks(ticks=None, labels=None, **kwargs)

  1. ticks：设置的坐标轴刻度，其他内容可以用文本属性可用于控制
  2. labels：设置刻度标签，设置同上文设置标签
import matplotlib.pyplot as plt
import random

fig = plt.figure(figsize = (12, 7), dpi = 100)

xpoints = range(120)
ypoints = [random.random()*20 + 10 for i in range(120)]

plt.plot(xpoints, ypoints)

# 自定义标签：range前闭后开
list_min = [str(i) if i > 9 else '0' + str(i) for i in range(60)]
xticks = ['10:{}'.format(i) for i in list_min]
xticks += ['11:{}'.format(i) for i in list_min]

plt.xticks(range(0, 120, 5), xticks[::5], rotation = 45)
plt.yticks(range(10, 32, 3))

plt.show()

2.3 显示图例

显示图例：matplotlib.pyplot.legend(*args, **kwargs)

  1. handles、labels：设置显示的图例标签
  2. loc：设置图例位置，有效值：’upper left’, ‘upper right’, ‘lower left’, ‘lower right’, ‘upper center’, ‘lower center’, ‘center left’, ‘center right’ 默认值为 ‘best’，根据内容自行指定
  3. ncol：设置图例的列数，默认为1(1列)；fontsize：设置图例字体大小；title：设置图例的标题
import matplotlib.pyplot as plt
fig = plt.figure(figsize = (12, 7), dpi = 100)

xpoints = [1, 2, 3, 4, 5, 6]
ypoints_1 = [12, 14, 15, 17, 20, 25]
ypoints_2 = [14, 12, 25, 17, 20, 15]

plt.plot(xpoints, ypoints_1, 'r--s')
plt.plot(xpoints, ypoints_2, 'b--s')
x_labels = ['red', 'blue']

# plt.plot(xpoints, ypoints_1, 'r--s', label = 'red')
# plt.plot(xpoints, ypoints_2, 'b--s', label = 'blue')

# prop 设置字体
plt.legend(labels = x_labels, loc = 'upper right', ncol = 1, fontsize = 12, title = "Color")
plt.show()

2.4 绘制网格

绘制网格：matplotlib.pyplot.grid(visible=None, which=’major’, axis=’both’, **kwargs)

  1. visible：是否可见，有效值：True、Flase、None(显示网格线)
  2. which：设置是否应用更改的网格线，有效值：’major’(表示应用更改的网格线), ‘minor’, ‘both’，默认为 ‘major’
  3. axis：设置展示网格线内容，有效值：’both’, ‘x’(仅显示 x轴), ‘y’(仅显示 y轴)，默认为’both’
  4. **kwargs：通过关键字方式指定 Line2D properties属性，详细内容见绘图3
import matplotlib.pyplot as plt
fig = plt.figure(figsize = (12, 7), dpi = 100)

xpoints = [1, 2, 3, 4, 5, 6]
ypoints = [12, 14, 15, 17, 20, 25]

plt.plot(xpoints, ypoints)
matplotlib.pyplot.grid(visible = True, which = 'both', axis = 'y', color = 'r', linestyle = '-', linewidth = 2)

plt.show()

2.5 绘制多图

绘制多图：

  1. matplotlib.pyplot.subplot(*args, **kwargs)：在绘图时需要指定位置
    • *args：包含三个参数：nrows(划分行数), ncols(划分列数), index(划分索引数，从 0 开始，按照从左到右，从上到下的顺序增加索引)
    • **kwargs：projection：统计图类型None, ‘aitoff’, ‘hammer’, ‘lambert’, ‘mollweide’, ‘polar’, ‘rectilinear’，默认为 rectilinear 折线图
# subplot()
import matplotlib.pyplot as plt
fig = plt.figure(figsize = (12, 7), dpi = 100)

xpoints = [1, 2, 3, 4, 5, 6]
ypoints_1 = [12, 14, 15, 17, 20, 25]
ypoints_2 = [14, 12, 25, 17, 20, 15]

plt.subplot(1, 2, 1, projection = 'rectilinear')            # 先指定第一行第一列，序号为1的子图，再进行绘图
plt.plot(xpoints, ypoints_1)
plt.title("plot_1")

plt.subplot(1, 2, 2)            # 指定第一行第二列，序号为2的子图；plt.subplot(2, 2, 3)绘制第二行第一列，序号为3的子图
plt.plot(xpoints, ypoints_2)
plt.title("plot_2")

plt.suptitle("subplot test")    # 针对多图进行命名
plt.show()
  1. matplotlib.pyplot.subplots(nrows=1, ncols=1, *, sharex=False, sharey=False, squeeze=True, subplot_kw=None, gridspec_kw=None, **fig_kw)：可以一次生成多个，在调用时只需要调用生成对象的 ax
    • nrows、ncols：设置图表的行数、列数，默认为 1
    • sharex、sharey：设置 x、y 轴是否共享，有效值：’none’/True, ‘all’/False, ‘row’(共享x轴), ‘col’(共享y轴)
# subplots()
import matplotlib.pyplot as plt
fig = plt.figure(figsize = (12, 7), dpi = 100)

xpoints_1 = [1, 2, 3, 4, 5, 6]
ypoints_1 = [12, 14, 15, 17, 20, 25]
xpoints_2 = [3, 4, 5, 6, 7, 8]
ypoints_2 = [14, 12, 25, 17, 20, 15]

plt.subplots(1, 1, sharex = "col")            # 先指定第一行第一列，序号为1的子图，再进行绘图
plt.plot(xpoints_1, ypoints_1)
plt.title("subplots_1")

plt.subplots(1, 1, sharex = "col")            # 指定第一行第二列，序号为2的子图；plt.subplot(2, 2, 3)绘制第二行第一列，序号为3的子图
plt.plot(xpoints_2, ypoints_2)
plt.title("subplots_2")

plt.show()

3. 其他图形

涉及到标签、刻度、图例等都是通用的；绘制图形参考

3.1 绘制散点图

绘制散点图：

matplotlib.pyplot.scatter(x, y, s=None, c=None, marker=None, cmap=None, norm=None, vmin=None, vmax=None, alpha=None, linewidths=None, *, edgecolors=None, plotnonfinite=False, data=None, **kwargs)

  1. x, y：传入两个参数，分别指定x、y轴的坐标，要求坐标数据数量必须一致；不能缺省 x轴 坐标
  2. s：点的大小，默认20；c：点的颜色，默认蓝色’b’，可以传入数组；marker：点的样式，默认为’o’
  3. cmap：Colormap，标量或者是一个 colormap 的名字；norm：Normalize，数据亮度在 0-1 之间，只有 c 是一个浮点数的数组的时才使用
  4. vmin，vmax：亮度设置，在 norm 参数存在时会忽略
  5. alpha：透明度设置，0-1 之间，默认为1或None(不透明)；linewidths：标记点的长度
  6. edgecolors：颜色或颜色序列，有效值：’face’, ‘none’, None，默认为 ‘face’
  7. plotnonfinite：布尔值，设置是否使用非限定的 c ( inf, -inf 或 nan) 绘制点
# 1，2，5：x,y,s,c,marker,alpha,linewidths
import matplotlib.pyplot as plt
fig = plt.figure(figsize = (12, 7), dpi = 100)

xpoints = [1, 2, 3, 4, 5, 6]
ypoints_1 = [12, 14, 15, 17, 20, 25]
ypoints_2 = [14, 12, 25, 17, 20, 15]

plt.scatter(xpoints, ypoints_1, s = 50, c = 'r')
plt.scatter(xpoints, ypoints_2, s = (20,50,100,500,2000,90), c = ('r', 'b', 'c', 'y', 'k', 'g'), marker = '^', alpha = 0.3, linewidths = 10)
plt.title("scatter")

plt.show()
# cmap
import matplotlib as mpl
import matplotlib.pyplot as plt
fig = plt.figure(figsize = (12, 7), dpi = 100)

xpoints = [1, 2, 3, 4, 5, 6]
ypoints = [12, 14, 15, 17, 20, 25]

# 需要生成 Normalize 实例
norm = mpl.colors.Normalize(vmin=-1, vmax=1)

plt.scatter(xpoints, ypoints_2, c = (20,50,100,500,2000,90), cmap = 'afmhot')
# cmap：可取 'viridis'(默认), 单色：'YlOrRd', 'Greens', 'Blues', 'Greys', 'Purples'; 拼色：'afmhot', 'cividis', 'inferno', 'magma', 'plasma'
plt.colorbar()    # 显示颜色条样例
plt.show()

3.2 绘制柱状图

柱状图：

绘制竖直型柱状图：matplotlib.pyplot.bar(x, height, width=0.8, bottom=None, *, align=’center’, data=None, **kwargs)
绘制水平型柱状图：matplotlib.pyplot.barh(y, width, height=0.8, left=None, *, align=’center’, **kwargs)

  1. x, height（width）：传入两个参数，分别指定x、y轴的坐标
  2. width（height）：设置柱状图宽度
  3. bottom：y轴底部坐标，默认为0；left：x轴底部坐标，默认为0
  4. align：柱形图与 x 坐标的对齐方式，有效值：’center’(默认，中心对齐), ‘edge’(左/下边缘对齐，可传递负宽度值实现右/上边缘对齐)
  5. **kwargs：包含color、edgecolor、linewidth、tick_label等
import matplotlib.pyplot as plt
fig = plt.figure(figsize = (12, 7), dpi = 100)

xpoints = [1, 2, 3, 4, 5, 6]
ypoints = [12, 14, 15, 17, 20, 25]

plt.subplot(1, 2, 1)
plt.bar(xpoints, ypoints, width = -0.5, bottom = 10, align = "edge")

plt.subplot(1, 2, 2)
plt.barh(xpoints, ypoints, height = 0.5, left = 1, align = "edge", color = ('r', 'b', 'c', 'y', 'k', 'g'))

plt.show()
# 绘制多条柱
import matplotlib.pyplot as plt
fig = plt.figure(figsize = (12, 7), dpi = 100)

xpoints = [1, 2, 3, 4, 5, 6]
ypoints_1 = [12, 14, 15, 17, 20, 25]
ypoints_2 = [14, 12, 25, 17, 20, 15]

plt.bar(xpoints, ypoints_1, label = 'blue', width = -0.3, bottom = 10, align = "edge")
plt.bar(xpoints, ypoints_2, label = 'orange', width = 0.3, bottom = 10, align = "edge")

plt.legend()

plt.show()

3.3 绘制饼图

绘制饼图：

matplotlib.pyplot.pie(x, explode=None, labels=None, colors=None, autopct=None, pctdistance=0.6, shadow=False, labeldistance=1.1, startangle=0, radius=1, counterclock=True, wedgeprops=None, textprops=None, center=(0, 0), frame=False, rotatelabels=False, *, normalize=True, data=None)

  1. x：表示每个扇形的面积
  2. explode：表示扇形到原点的距离，以半径的倍数计算，默认值为0(紧贴圆心)，传入一个数组；labels：各个扇形的标签，默认值为 None；colors：表示各个扇形的颜色，默认值为 None，会自动指定；autopct：设置饼图内各个扇形百分比显示格式，”%0.2f%%” 两位小数百分比
  3. pctdistance：设置百分比到圆心的距离，默认为0.6倍半径；labeldistance：绘制标签位置，默认值为1.1倍半径；shadow：设置是否阴影，默认为 False；startangle：其实绘制角度，默认为0(x轴正半轴)；radius：饼图的半径，默认为1；counterclock：设置绘制方向，默认为 True(逆时针)
  4. wedgeprops：传递给 wedge 对象用来画一个饼图，传递一个字典，如：wedgeprops={‘linewidth’:5}；textprops：用于设置标签和比例文字的格式，传递一个字典
  5. center：设置图表中心的坐标(需要和 frame 配合使用)；frame：是否绘制带有坐标轴的图；rotatelabels：旋转标签值同一角度
import matplotlib.pyplot as plt
fig = plt.figure(figsize = (12, 7), dpi = 100)

from pylab import mpl
mpl.rcParams["font.sans-serif"] = ["SimHei"]

xpoints = [12, 14, 15, 17, 20, 25]
x_labels = ['1月', '2月', '3月', '4月', '5月', '6月']

plt.pie(xpoints, explode = (0.2, 0, 0, 0, 0, 0), labels = x_labels, colors = ('r', 'b', 'c', 'y', 'k', 'g'), autopct = "%0.2f%%", 
        pctdistance = 0.8, labeldistance = 1.2, shadow = True, startangle = 90, radius = 2.2, counterclock = False, center = (2, 2), 
        frame = True, rotatelabels = True)

plt.legend(labels = x_labels, loc = 'upper center', ncol = 2, fontsize = 12, title = "Month")   # 显示图例，详情见2.4
plt.show()

3.4 绘制直方图

绘制直方图：只有为经过统计处理的原始数据才能回执直方图

matplotlib.pyplot.hist(x, bins=None, range=None, density=False, weights=None, cumulative=False, bottom=None, histtype=’bar’, align=’mid’, orientation=’vertical’, rwidth=None, log=False, color=None, label=None, stacked=False, *, data=None, **kwargs)

  1. x：直方图绘制的数据；bins：直方图绘制的列数，默认为 10；range：规定 x轴 的最大值最小值元组
  2. density：设置显示密度；cumulative：进行计算绘制，第n条柱的值为前n条柱的和；bottom：每一条柱的y轴起始位置，可以对每一条柱设置值，默认为0
  3. histtype：数据展示类型，有效值：’bar’(柱形), ‘barstacked’(), ‘step’(只显示边框，不填充), ‘stepfilled’(填充)，默认值为 ‘bar’；align：设置直方图条的水平对齐方式，有效值：’left’, ‘mid’, ‘right’，默认为 ‘mid’；orientation：设置图条朝向，有效值：’vertical’(竖直), ‘horizontal’(水平)，默认为’vertical’
  4. rwidth：设置柱形宽度，在 histtype = ‘step’/‘stepfilled’时无法使用；color：设置柱形颜色
import matplotlib.pyplot as plt

fig = plt.figure(figsize = (12, 7), dpi = 100)

plt.hist(xpoints, bins = 8, density = True, bottom = 0.1, cumulative = True, histtype = 'step', align='right', orientation='horizontal',
