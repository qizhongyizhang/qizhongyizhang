    • interface

    • if
    • else
    • do

    • switch
    • case

    • break：主要用在循环语句或者 switch 语句中，用来 跳出最里层的循环，并且继续执行该循环下面的语句
    • continue：用于任何循环控制结构中，用于让程序立刻跳转到下一次循环的迭代：在 for 循环中，continue 语句使程序立即跳转到更新语句；在 while 或者 do…while 循环中，程序立即跳转到布尔表达式的判断语句

2. 数据类型

2.1 基本数据类型

数值型

  • 整数：存放整数
    • byte[1]：字节
    • short[2]：短整型
    • int[4]：整数型(默认)
    • long[8]：长整型，申明时需要加上“i”或者“L”
  • 浮点数：存放小数，由符号位、指数位和尾数位构成，尾数位可能丢失，存在精度问题，有十进制计数法和科学计数法两种表现形式
    • float[4]：单精度浮点数，申明时需要加上“f”或“F”
    • double[8]：双精度浮点数(默认)
      不要使用运算结果为小数进行相等判断，通常采用两数之差的绝对值小于某个精度范围就认为相等

字符串类型：char[2]，存放单个字符，可用ASCLL码/Unicode码输出字符，输出的字符用单引号包裹，双引号包裹的是字符串，char字符都有Unicode码对应的整数，可以参与运算

布尔类型 ：boolean[1]，存放ture，false，常用于逻辑运算，进行流程控制，不能用0或非0的整数代替true或false

数据类型转换：小容量数据类型可以自动转换为大容量类型

自动类型转换：存在多种类型的数据混合运算时，先自动转换为容量最大的数据类型

byte、short和char不能自动进行类型转换，byte、short和char进行运算(包括单类型运算和多类型运算)时先自动转换为int进行计算
boolean 不参与数据类型转换
当把整数赋值给byte时，先判断整数是否在byte范围内，在进行类型判断进行赋值，如果是变量进行赋值，则先进行类型判断

强制类型转换：用强制类型转换符()包裹要转换成的数据类型，后面接上要转换的数据，可能引起数据精度丢失和数据溢出的问题

强制类型转换符只针对最近的操作数有效，可以使用()提高运算优先级
char可以保存int值，不能保存int的变量，需要进行强制类型转换符

String类型转换：仅能转换整数字符串为基本数据，不能转换字符字符串为基本数据

基本类型转换为String数据类型：只需要在基本数据类型后面加上’ + “” ‘即可
String类型转换为基本数据类型：需要用到基本数据类型包装类的parseXxx方法，String转换为char类型是使用charAt()方法取字符串的第一个字符给char

3. 运算符

3.1 算术运算符

算术运算符：对数值数据的变量进行运算

运算符 运算 范例 结果
+ 正号 +1 1
- 负号 -1 -1
+ 加 1 + 1 2
- 减 1 - 1 0
* 乘 1 * 1 1
/ 除 1 / 1 1
% 取余 1 % 1 0
++ 自增（在前）：先运算再赋值
自增（在后）：先赋值再运算		 a = 2; b = ++ a
a = 2; b = a ++		 a = 3; b = 3
a = 3; b = 2
– 自减（在前）：先运算再赋值
自减（在后）：先赋值再运算		 a = 2; b = – a
a = 2; b = a –		 a = 1; b = 1
a = 1; b = 2

**+**：拼接，当左右两边数据类型均为数值型时，进行加法运算；若一边为字符串类型，进行拼接运算，运算顺序为从左到右
**/**：除，当除号两边均为int时，结果会舍弃小数部分，仅保留整数部分
**%**：取模，本质是：a % b = a - a / b * b
**++**：独立使用时，前++与后++完全一致，作为表达式使用：

int i = 1;
i++;            // 等价于：i = i + 1;
++i;            // 等价于：i = i + 1;

int j = ++i;    // 等价于：i = i + 1; j = i;
int j = i++;    // 等价于：j = i; i = i + 1;
i = i++;        // 得i = 1 ：(1)temp = i; (2)i = i + 1; (3)i = temp
i = ++i;        // 得i = 2 ：(1)i = i + 1; (2)temp = i; (3)i = temp
注：--同理

3.2 关系运算符

关系运算符：判断两数的大小关系，结果都是boolean类型；常用在 if 分支结构或 for / while 循环结构中，有关系运算符组成的表达式成为关系表达式

运算符 实例 描述
== 等于 1 == 1 true
!= 不等于 1 != 1 false
> 大于 1 > 1 false
< 小于 1 < 1 false
>= 大于等于 1 >= 1 true
<= 小于等于 1 <= 1 true
instanceof 检查是否为类的对象 “A” instanceof String true

3.3 逻辑运算符

逻辑运算符：用于连接多个条件(关系表达式)，最终结果也是一个boolean值

a b a&b a&&b a|b a||b !a a^b
true true true true true true false false
true false false false true true false true
false true false false true true true true
false false false false false false true false
  • **逻辑与 &**（and）：当运算符两边的表达式的结果都为true时，整个运算结果才为true，只要有一边为false，则结果为false
    • &：无论第一个判断条件是否成立，逻辑运算符前后两个条件都会进行判断
  • **短路与 &&**（and）：当运算符两边的表达式的结果都为true时，整个运算结果才为true，只要有一方为false，则结果为false
    • &&：第一个条件不成立，后面的条件均不进行判断，返回false，推荐使用，效率更高
  • **逻辑或 |**（or）：当运算符有一边为true时，整个运算结果为true
    • |：无论第一个判断条件是否成立，逻辑运算符前后两个条件都会进行判断
  • **短路或 ||**（or）：当运算符有一边为true时，整个运算结果为true
    • ||：第一个条件成立，后面的条件均不进行判断，返回true，推荐使用，效率更高
  • **逻辑非 !**（not）：取本来值的相反值
  • 逻辑异或 ^：符号两边不同时结果为true，否则为false

3.4 赋值运算符

赋值运算符：将数据赋值给变量，左边只能是变量，右边可以是常量、变量、表达式，进行复核赋值运算时会进行类型转换
基本赋值运算符：=，例：int a = 10;
复合赋值运算符：+=、-=、*=、/=、%=，例：a += b; 等价于 a = a + b;其他同理

3.5 三元运算符

条件表达式 ？ 表达式1 : 表达式2；若条件表达式为true，则运行表达式1；若条件表达式为false，则运行表达式2。通常可以转换为if ··· else ··· 语句
当条件表达式为true时，不会执行表达式2；当条件表达式为false时，不会执行表达式1

3.6 运算符优先级

(), [], {}等 -> 单目运算符 -> 算术运算符 -> 位移运算符 -> 比较运算符 -> 逻辑运算符 -> 三元运算符 -> 赋值运算符

优先级 运算方向 运算符
高 L->R () [] {}
R->L ++ – ~ !(data_type)：单目运算符，仅对一个数据进行计算
L->R * / % (算术运算)
L->R + -
L->R << >> >>> (位运算)
L->R < > <= >= instanceof (比较运算)
L->R == !=
L->R & (逻辑运算)
L->R ^
L->R |
L->R &&
L->R ||
L->R expr ? value : value (三元运算符)
R->L = *= /= %=
R->L += -= <<= >>=

4. 控制语句

4.1 循环语句

  • for(初始化循环变量; 判断条件; 循环变量增幅){循环体}
for(int i = 0; i < n; i++){
	System.out.print("i");
}

增强for循环：for(声明循环变量; 数组名称){语句}，主要用于数组

String[] str={'xx','xxx'}
for(String s:str){
	System.out.println(i);
}

注意：增强for的目标不能是null（使用前先做非空判断），声明局部变量的类型必须和数组元素的类型匹配

  • while(条件){语句}
while(条件){
	System.out.print("i");
}
  • do {语句}while(条件);
do{
	System.out.print("i");
}while(条件)

比较：与while相比，do…while 循环至少会执行一次；for的循环次数提前已知

4.2 分支语句

  • if(条件){语句}[else if(条件){语句}else{语句}]
if(){
	System.out.print("i");
}
else if(条件){
	System.out.print("j");
}
else{
	System.out.print("k");
}
  • switch(变量) {case value:{语句;break;}[default:{语句}]}
switch(变量){
	case value1:{
		System.out.print("i");
		continue;
	}
	case value2:{
		System.out.print("j");
		break;
	}
	default:{
		System.out.print("k");
	}
}

switch的类型可以是int，short，byte和char，不能处理long，String类型；匹配之前不会执行，若匹配无break，则后续会继续执行

比较：switch使用了Binary Tree算法，运行效率优于if（除非是if-else的第一个条件就为true），if的使用更加灵活，条件更为多样；编译器编译switch时，不管有多少case，都直接跳转，不需逐个比较查询，与if不同；分支较多时，优先考虑switch

5. I/O

5.1 Input(输入)

Scanner

  • 导包：import java.util.Scanner
  • 输入：Scanner sc = new Scanner(System.in);

5.2 Output(输出)

Java的输出方法是在java.long包里的System类中的方法，有print、println、printf三种

  • System.out.print()；是一般的标准输出，输出的信息不会换行
  • System.out.println()；与print功能相似，输出的信息后会换行
  • System.out.printf()；继承了C语言的printf的一些特性，可以进行格式化输出
    符号 说明 适用范围 备注
    %d 以十进制整数格式输出 byte、short、int、long、Byte、Short、Integer、Long、BigInteger 无
    %f 以十进制浮点数格式输出 float、double、Float、Double、BigDecimal 无
    %e或%E 以科学记号浮点数格式输出 float、double、Float、Double、BigDecimal %e表示以小写字母e表示，%E表示以大写字母E表示
    %o 以八进制整数格式输出 byte、short、int、long、Byte、Short、Integer、Long、BigInteger 无
    %x或%X 以十六进制整数格式输出 byte、short、int、long、Byte、Short、Integer、Long、BigInteger %x表示以小写字母x表示，%X表示以大写字母X表示
    %c或%C 以字符符号输出 byte、short、char、Byte、Short、Integer、Charcater %c表示以小写字母表示，%X表示以大写字母表示
    %b或%B 输出boolean值 boolean、非null值为true、null值为false %b表示输出结果会是true或false，%B表示输出结果会是TRUE或FALSE
    %h或%H 用Integer.toHexString (arg.hashCode())来得到输出结果 - 如果arg是null，则输出null，也常用于想得到十六进制格式输出
    %n 输出平台特定的换行符号 - Windows下会置换为”\r\n”；Linux下会置换为’\n’；Mac OS下回置换为’\r’

6. 简单函数

进制转换函数：

10进制的x转2进制:   Integer.toBinaryString(x);   # 返回一个二进制int型数据
10进制的x转8进制:   Integer.toOctalString(x);    # 返回一个八进制int型数据
10进制的x转16进制:  Integer.toHexString(x);      # 返回一个16进制int型数据
10进制的x转n进制:   Integer.toString(x,n);       # 返回一个n进制int型数据
n进制的x转10进制:   Integer.valueOf(x,n);        # 返回一个十进制int型数据；valueOf()：用于返回给定参数的原生Number对象值的静态方法，参数可以是原生数据类型, String等。该方法接收两个参数一个是需要转换字符串[，一个是转换基数(可选)]
Number.valueOf("444"/444[,16]);     # 返回字符"444"/数值444对应的十六进制Number(Integer/Float/Double/String)型数据

n进制的x转m进制(n->10->m)：（2进制的10001110转8进制）
Integer.toOctalString(Integer.valueOf(10001110, 2));
BigInteger.change("10001110", 2, 8);    # BigInteger.change(String number/"x", int from/n, int to/m);

字符串相关函数

  • 字符串截取：substring(start, stop); (包头不包尾)

方法重载：方法名相同，可以改变参数类型、参数个数、参数顺序，仅改变参数名不构成方法重载，方法重载使用同一个方法名实现功能相同、主体不同的方法
