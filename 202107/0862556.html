写在前面：本文主要记录 Python使用 的相关基础知识

1. 保留关键字

保留关键字：36个（Python 3.9），已经或者将要被赋予特定意义的单词，不能使用这些保留字作为标识符给变量、函数、类、模板以及其他对象命名

查看关键字: 关键字在不停的完善，可能更高的版本会有修改
import keyword
print(keyword.kwlist)

1.1 控制语句

引用控制：import（导入文件、外部库、模块）、from
循环结构：while、for、break、continue
分支结构：if、elif、else
异常控制：try、except、finally、raise（抛出异常）、assert（设置断言，用来辅助查看，用于调试）

1.2 变量修饰符

class：定义类
def：定义函数
lambda：匿名函数
return：定义返回值
global：申明全局作用域
nonlocal：申明外部嵌套函数内的变量

1.3 逻辑判断

True：布尔是
False：布尔非
and：逻辑与
not：逻辑非
or：逻辑或
is：身份运算符，比较两个对象的内存地址是否一致，也可用 is not；is 判断两个变量的内存地址是否一致，== 判断引用变量的值是否一致
in：成员运算符，是否在某个范围内

1.4 其他

__peg_parser__
as：给模块、变量等起别名，方便使用
async
await
del：删除一个对象
None：空值
pass：表示占位符，保证程序代码的结构正确
with
yield：生成器返回申明

注：没有数据类型关键字，在赋值时自动推导确定对应变量的数据类型，支持数值类型有：整数类型（int）、浮点数类型（float）、复数类型（complex）、字符串（没有char保留字）、集合（set）、列表（list）、字典（dict，也叫映射）；随时命名、随时赋值（=）、随时使用

2. 运算符

2.1 算术运算符

算术运算符：对数值数据的变量进行运算

运算符 实例 描述
+ 加 1 + 1，结果为2；可以处理两个字符串表示拼接字符串：’y’ + ‘es’，输出’yes’
- 减 1 - 1，结果为0
* 乘 1 * 1，结果为1；可以和字符串配合，表示输出整数次字符串：’s’ * 2，输出两个 ‘ss’
/ 除 1 / 1，结果为1
// 求模（整数商） 1 // 1，结果为1
% 取余 1 % 1，结果为0
** 幂 1 ** 1，结果为1

2.2 比较运算符

比较运算符：判断两数的大小关系

运算符 实例 描述
== 等于 左右两边是否相等，是返回True
!= 不等于 左右两边是否不等，是返回True；Python2可用 <> 表示不等于
> 大于 左边是否大于右边，是返回True
< 小于 左边是否小于右边，是返回True
>= 大于等于 左边是否大于等于右边，是返回True
<= 小于等于 左边是否小于等于右边，是返回True

2.3 逻辑运算符

逻辑运算符：

  • and：当运算符两边的表达式的结果都为true时，整个运算结果才为true，只要有一边为false，则结果为false
  • or：当运算符有一边为true时，整个运算结果为true
  • not：取本来值的相反值

2.4 赋值运算符

赋值运算符：

运算符 实例 描述
= 赋值 c = a + b
+= 加赋值 c += a；c = c + a；对于列表则是调用 extend 方法
-= 减赋值 c -= a；c = c - a
*= 乘赋值 c *= a；c = c * a
/= 除赋值 c /= a；c = c / a
//= 求模赋值 c //= a；c = c // a
%= 取余赋值 c %= a；c = c % a
**= 幂赋值 c **= a；c = c ** a

转义字符：特殊含义转义

\\, \', \" ：转义，输出 \, ', "
\n：换行，显示 \n 需要写成 \\n，即进行转义
\t：制表位
\r：回车
r'' 中包裹的字符不进行转义，print(r'\\\t\\')，输出：\\\t\\

3. 输入输出

  • **input()：接收输入，input([输入提示])**，默认返回对应的字符串
str = input('Please entry:')  # 用变量 str 接收来自键盘的输入

num = eval(input('Please entry:'))   # 通过 eval() 将字符串行的数字转换为数值型，将字符串当成一个有效的表达式来求值，并返回计算结果，不建议使用，可以通过输入 __import__('os').system('终端命令')进行攻击：__import__('os').system('rm -rf')
num = int(input('Please entry:'))    # 通过 int() 将字符串行的数字转换为整数型
num = float(input('Please entry:'))  # 通过 float() 将字符串行的数字转换为浮点型
  • **print()：显示输出，print(打印内容 [, sep = ‘ ‘, end = ‘\n’])**，多个打印内容书写时用 , 隔开；打印时以 sep 隔开，默认为空格；以\n结束
print('hello', 'world')
hello world

print(100 + 200)  # 可计算结果
300

print('100 + 200 =', 100 + 200)  # 100 + 200 = 用 '' 包裹，表示字符串，而非表达式
100 + 200 = 300

格式化输出：

  1. 占位符：print(“XXX 占位符 “%(value1,value2))

    %s：字符串类型占位符
%d：整数类型占位符（直接截取整数位，%04d可表示，数字宽为4位，不足用0进行左填充）
%f：浮点型占位符
%x：16进制数占位符

print('Age: %d, Gender: %s, growth rate: %.2f %%' % (25, '男', 7.5))，% 后面的数据是一个元组

tuple_info = (25, '男', 7.5)
print('Age: %d, Gender: %s, growth rate: %.2f %%' % tuple_info)
Age: 25, Gender: 男, growth rate: 7.50 %
输出内容中有 % 号时，需要用两个 %% 表示

  2. format：print(‘XXX {字符填充序号 : [填充字符] [对齐方式] [数据宽度] [千位分隔符] [保留小数位数] [数据类型]}’.format(value1,value2))

字符填充序号：format() 里面的数据索引，从零开始
填充字符：数据长度不够时的填充字符，默认用空格进行填充
对齐方式：数据对齐方式，^, <, > 分别是居中、左对齐、右对齐
数据宽度：数据的实际显示长度，用正整数表示，默认宽度为10，数据超过宽度则省略后宽度之后的数据
千位分隔符：数据级每隔 1000 的分隔标识符
保留小数位数：用 .+正整数表示，如 .4 表示保留 4位整数
数据类型：d 表示整数，f 表示浮点数，s 表示字符串
>>> print('{0:@<20,.4f}'.format(123456789.987654321))
123,456,789.9877@@@@

显示百分比：print('{0:-<30,.4%}'.format(123456789.987654321))
12,345,678,998.7654%----------
科学计数法：print('{0:-<30,.4e}'.format(12345646577886789.987654321))
1.2346e+16--------------------

二进制：'{:b}'.format(11)
十进制：'{:d}'.format(11)
八进制：'{:o}'.format(11)
普通十六进制：'{:x}'.format(11)
小写十六进制带b：'{:#x}'.format(11)
大写十六进制带B：'{:#X}'.format(11)
  1. f-string：**var定义（换行）print(f’XXX {var_name[:format格式]}’)**，使用 f 开头申明，用前面申明的对应变量替换相同位置的占位符
    >>> r = 2.5
>>> s = 3.14 * r ** 2
>>> print(f'The area of a circle with radius {r} is {s:.2f}.')
The area of a circle with radius 2.5 is 19.62.

4. 其他

标识符：程序员定义的变量名、函数名、类名等

标识符命名规范

由字母、下划线、数字组成，首字符不能是数字，可以使用汉字但不推荐；不能和关键字同名
严格区分大小写，可以采用下划线命名（用下划线 _ 分割单词）、大驼峰（不同单词首字母大写）、小驼峰（第一个单词首字母小写，其他单词首字母大写）等命名法；可以使用保留关键字的大写形式命名，但不推荐
常用全部大写的变量名表示常量

注释：对某些代码进行标注说明，增强代码的可读性

  • 单行注释：# 注释内容
  • 多行注释：’’’ 注释内容 ‘’’/“”” 注释内容 “””
  • 文档注释：
class Person(object):
    """
    Person类
    """
    def __init__(self, name, weight):
        """
        person类的初始化方法
        :param name: 姓名
        :param weight: 体重
        """
        self.name = name
        self.weight = weight


# 输出注释文档
if __name__ == "__main__":
    import test            # 要与文件名保持一致
    print(help(test))
  • TODO注释：表明需要做的工作
