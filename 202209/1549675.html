数据分析工具-SPSS安装使用

写在前面：本文主要记录 数据分析工具-SPSS安装使用 的相关基础知识

1. 介绍

1.1 发展

SPSS：Statistical Product and Service Solutions，世界上最早的统计分析软件，已被IBM收购；可以跟python、R等连接使用；xxx.sav 为文件存储名称；[数据集] 为文件工作名称
60年代：美国斯坦福三位研究生研发
70年代：1975年成立法人组织、在芝加哥组建了SPSS总部，退出SPSS中小型机版-SPSSX
80年代：微机版(V14)SPSS/PC+
90年代：Windows版(V610)
本世纪：11~25，中文版，2010年被IBM收购

注：SAS：统计分析软件，有一定难度，难上手，收费

1.2 优缺点

优点：

  • 功能强大
  • 兼容性好
  • 易用性强，是应用统计人员的首选
  • 扩展性高（以一种不同的方式）

缺点：

  • 计算速度相对较慢
  • 在统计模型的纳入上速度较慢

2. 使用

四大窗口：

数据窗口：包括数据视图和变量视图（标签表示字段描述，值表示数据字典）

输出窗口：分析 -> 描述统计 -> …

语法窗口：文件 -> 新建/打开 -> 语法，输入语法 -> 点击运行，查看器窗口查看运行结果

脚本窗口：文件 -> 新建/打开 -> 脚本，选择 Python、Basic等

五个标准按钮：

  • 确定、取消
  • 粘贴：用于自动生成SPSS程序
  • 重置：恢复对话框为初始状态
  • 帮助：方法，用法介绍

变量存储类型：

  • 数值型：应用最为广泛
  • 字符型：由于分析、整理都较困难，建议少量使用，改为编码录入；可通过值标签进行字符和字典的切换
  • 日期型：实际是特殊的数值型变量，尽量少用

标量：连续的数值型变量

有序：有序的分类标签

名义：无序的分类标签

2.1 数据I/O

2.1.1 导入

Excel：点击文件 -> 选择导入数据 -> 选择Excel -> 在弹出的对话框中选择文件打开位置 -> 选择文件类型为Excel -> 选择需要打开的数据文件 -> 选择需要打开的工作表名称 -> 点击确定

MySQL：点击文件 -> 选择导入数据 -> 选择数据库 -> 选择新建查询 -> 添加 ODBC 的数据源 -> 配置连接信息和数据库信息 -> 点击确认 -> 选择数据表 -> 点击确认

文本数据：点击文件 -> 选择导入数据 -> 选择文本数据 -> 选择需要打开的数据文件 -> 点击打开，进入文本导入向导 -> 默认下一步 -> 默认下一步 -> 默认下一步（SPSS中一条数据成为个案） -> 默认下一步 -> 针对每一个变量（SPSS中称数据字段为变量）选择合适的数据格式，点击下一步 -> 勾选粘贴语法，点击完成 -> 在弹出的语法编辑器中运行粘贴出来的语法，才能完成文本数据的导入

2.1.2 导出

导出数据视图：点击文件 -> 选择导出数据 -> 选择导出数据数据类型 -> 可选择导出的变量列表，也可以勾选将工作表追加到现有文件中 -> 选择导出位置 -> 点击将文件存储到存储库

2.2 操作

新增录入：在变量视图输入新增的变量，按 Enter 可以直接自动识别对应的数据类型

查找替换：Ctrl + F 查找时会自动筛选列

计算变量：点击转换 -> 选择计算字段 -> 在弹出的计算变量对话框输入计算算法（支持新增变量和原变量计算） -> 点击确认，完成数据计算

排序

  • 个案排序：点击数据 -> 选择个案排序 -> 在弹出的对话框中进行排序，支持多变量不同排序方式
  • 变量排序：点击数据 -> 选择变量排序 -> 选择排序方式，实现变量排序

2.3 数据管理

2.3.1 数据选择

数据拆分：点击数据 -> 选择拆分文件 -> 在弹出的对话框中勾选比较组，选择分组依据 -> 再进行分析的时候会按照分组变量进行分组统计分析；分析结束后需要取消拆分，否则后续统计分析会默认进行分组

数据筛选：点击数据 -> 选择 选择个案 -> 在弹出的对话框中勾选 如果条件满足… -> 在弹出的对话框中输入筛选条件，可以完成筛选；分析结束后需要取消筛选，否则后续统计分析会默认进行筛选

2.3.2 数据汇总

数据汇总：点击数据 -> 选择 汇总 -> 在弹出的对话框中选择分解变量（分组变量）、变量摘要（需要汇总展示的变量列表），可针对每一个变量指定数据汇总方式

数据 -> 重构（一条数据转化成多条数据）

合并文件：数据 -> 合并文件 -> 添加个案，可针对列名不一致的内容可以进行配对

标识重复个案：数据 -> 标识重复个案 -> 在 定义匹配个案的依据中选择判定重复的变量

2.4 图表

图形 -> 图标构建器，采用拖动方式实现图标构建
