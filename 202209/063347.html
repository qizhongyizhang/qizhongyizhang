Excel进阶-公式与函数

写在前面：本文主要记录 Office-Excel进阶-公式与函数 的相关基础内容

1. 简介

公式：以=开头的，对地址进行引用的计算形式

函数：是Excel 预定义的一种内置公式；通过参数来按特定的顺序或结构执行计算

1.1 地址引用

地址引用：

  • 相对引用：范围拉动时会随着拉动不断改变；A2：A5

  • 绝对引用：范围拉动时位置不会改变；$A$2；可以通过 F4 快速添加 $ 符号

  • 混合引用：设置改变范围；$A2：拉动改变行数，$A3、$A4等；A$2：拉动改变列数，B$2、C$2等

1.2 运算符

算术运算符：对数值数据的变量进行运算

运算符 实例 描述
+ 加 1 + 1，结果为2
- 减 1 - 1，结果为0
* 乘 1 * 1，结果为1
/ 除 1 / 1，结果为1
% 百分比 1 % 1，结果为100%
^ 幂 1 ** 1，结果为1

关系运算符：判断两数的大小关系

运算符 实例 范例 结果
= 等于 1 = 1 true
<> 不等于 1 != 1 false
> 大于 1 > 1 false
< 小于 1 < 1 false
>= 大于等于 1 >= 1 true
<= 小于等于 1 <= 1 true

2. 函数

2.1 文本函数

len(text) ：返回指定文本字符串 text 中字符的个数（文本长度），不分中英文和数字，都是1个字符，=len(A1)
left(text [, num_chars]) ：从文本字符串左边（开头）起第一个字符开始返回指定个数的字符，=left(A1, 3)；**right(text [, num_chars])**：从文本字符串右边（末尾）起第一个字符开始返回指定个数的字符，=right(A1, 3)

text：文本内容
[num_chars] ：指定提取字符的个数，默认值为1；可选

**mid(text, start_num, num_chars)**：从文本字符串的指定位置提取指定长度的字符，=mid(A1, 3, 5)

text：文本内容
start_num：开始提取的位置
num_chars：提取的字符数

**text(内容, 格式)**：将给定的内容按照特定格式进行转换，=text(A1, “yyyy-mm-dd”)；指定格式可以查看单元格中数字的自定义格式模板

**replace(old_text, start_num, num_chars, new_text)**：根据指定的内容，将原文本部分内容替换成新的内容，=replace(A1, 1, 3, “*“)：将A1单元格从第1位开始后面3位的内容替换为 *；

old_text ：原来的文本
start_num：开始替换的位置
num_chars：替换的字符数
new_text：替换的新内容

**find(find_text, within_text [, start_num])**：根据指定内容查找，返回查找文本所在位置的起始值，=find(“f”, A1, 2)：从A1单元格第2位开始查找 f

find_text: 要查找的文本
within_text：查找的范围
start_num：指定开始查找的位置，可选

2.2 统计函数

**int(number)**：向下舍入取整（仅保留整数部分），number一般是个小数，正负都可以；=int(5.7)，返回5；=int(-5.7)，返回-6

**round(number, num_digits)**：将数字四舍五入到指定的小数位；=round(3.1415926, 3)，返回3.142

number ：需要进行四舍五入的数字
num_digits：进行四舍五入运算的位置，即保留的小数位数

**mod(number, divisor)**：计算两数相除的余数；=mod(10, 3)，返回1

number：被除数
divisor：除数

**count(number1 [, number2, …])**：计数；count(A1:A10)
**max(number1 [, number2, …])**：求最大值；max(A1:A10)
**min(number1 [, number2, …])**：求最小值；min(A1:A10)
**sum(number1 [, number2, …])**：求和；sum(A1:A10)
**average(number1 [, number2, …])**：求平均数；average(A1:A10)
**countif(range, criteria)**：条件计数；=count(A1:A10, “A”)：统计A1到A10取值为 “A” 的数量

range：统计非空单元格数目的区域
criteria：给定的条件

**countifs(criteria_range1, criteria1 [, criteria_range2, criteria2, …])**：多条件计数

criteria_range1 ：条件区域1
criteria1：条件1

**sumif(range, criteria [, sum_range])**：条件求和；=sumif(A1:A10, “>5”, B1:B10)，A1到A10取值为大于5时，计算对应B列的和

range：条件所在的数据区域
criteria：给定求和的筛选条件
sum_range：求和区域；若省略，则代表求和区域与条件所在区域一致

**sumifs(sum_range, criteria_range1, criteria1 [, criteria_range2, criteria2, …])**：多条件求和

sum_range：求和范围
criteria_range1：条件范围
criteria1：条件

2.3 查找与引用函数

**vlookup(lookup_value,table_array,col_index_num,range_lookup)**：查找匹配数据

lookup_value：想要查找的内容
table_array：想要查找的区域
col_index_num：指定返回的列号
range_lookup：指定匹配方式，是否近似匹配：1/True 表示近似匹配， 0/False 表示精确匹配

**match(lookup_value, lookup_array [, match_type])**：返回查找值所在区域的位置

lookup_value：查找的值
lookup_array：查找区域
match_type：查找方式：为1时，查找小于或等于lookup_value的位置；为0时，查找等于lookup_value的位置；为-1时，查找大于或等于lookup_value的位置；默认为 1

2.4 日期函数

today() ：返回当前的系统日期
now() ：返回当前系统的日期和时间
year(value) ：返回 value 所在的年份
month(value) ：返回 value 所在的月份
day(value) ：返回 value 所在的日
**date(year, month, day)**：拼接日期格式；=date(2022, 09, 07)，返回 2022/9/7，默认不会保留前面的0
**datedif(start_date, end_date, unit)**：计算日期之间的差值；

start_date：开始时间
end_date：结束时间
unit：计算单位，有效值 “Y”（年）、”M”（月）、”D”（日）

2.5 其他

2.5.1 逻辑函数

**if(logical_test,value_if_true,value_if_false)**：逻辑判断，根据判断结果返回对应结果

logical_test ：给定的判断条件
value_if_true ： 条件成立则返回的值
value_if_false ：条件不成立则返回的值

  1. 单一条件判断：if(A1 > 0, 1, -1) ：如果A1单元格的数值大于0，返回1；否则返回-1
  2. 多条件判断：if(A1 > 0, 1, if(A1 = 0, 0, -1)) ：如果A1单元格的数值大于0，返回1；等于0，返回0；小于0，返回-1
  3. 与and和or联合使用：if(and(A1 > 0, B1 = 1), 1, -1) ：如果A1单元格的数值大于0，并且B1单元格的数值等于0，返回1；否则返回-1
    • and(逻辑判断1，逻辑判断2…) 判断条件都为真时，返回True，否则False
    • or (逻辑判断1，逻辑判断2…) 判断条件存在真时，返回True，否则False

2.5.2 常见出错信息

**DIV/0!**：零作除数
**NAME?**：不能识别名称（删除了公式中引用的名称、使用不存在的名称或者函数名拼写错误）
**VALUE!**：不正确的参数或运算符
**REF!**：引用了无效的单元格地址（删除或移动了由其它公式引用的单元格）
**NULL!**：指定了两个并不相交的区域
N/A：当在函数或公式中引用了无法使用的数值
**NUM!**：数字类型不正确
**######**：列宽设置问题，不是错误值 ，输入到单元格中的数值太长，在单元格中显示不下
