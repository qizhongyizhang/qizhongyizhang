Excel进阶-数据透视表

写在前面：本文主要记录 Office-Excel进阶-数据透视表 的相关基础内容

1. 简介

数据透视表：把明细表进行分类汇总形成的汇总表

1.1 使用场景

  1. 有大量的数据，使用函数计算慢、繁琐
  2. 希望快速制作、整理、分析各类报表
  3. 数据源经常发生变化，并且要经常分析和处理最新的数据
  4. 想快速洞察数据背后隐藏的意义

1.2 使用条件

  1. 不能包含空字段：包含空字段，无法制作数据透视表
  2. 不包含相同字段 ：相同的字段名，会自动添加序号，以示区别
  3. 不能有合并单元格 ：字段所在行有合并单元格，等同于空字段，也无法创建数据透视表
  4. 不能有空行：如果有空行，会当成空值处理
  5. 不包含有文本型数字：文本型数字会导致汇总求和的结果错误

2. 使用

新建：点击 插入菜单栏 -> 点击 数据透视表 -> 选择合适的行标签和列标签 -> 选择列标签值汇总方式方式（包括：求和、计数、平均值、最值、乘积、标准差、方差等统计指标，默认为求和）和值显示方式（包括百分比、排序方式等）-> 点击确认，即可完成

2.1 更新数据源

  • 修改单个单元格：源数据表里面的数据更新后，进行数据刷新（默认源数据和数据透视表数据不会同步更新）；点击数据透视表，召唤上下文选项卡 -> 选择选择 -> 点击刷新，选择全部刷新，即可完成数据刷新
  • 源范围内新增行：源数据表里面的数据更新后，进行数据刷新；点击数据透视表，召唤上下文选项卡 -> 选择选项 -> 点击刷新，选择全部刷新，即可完成数据刷新
  • 源范围外新增行：源数据表里面的数据更新后，进行数据源更改；点击数据透视表，召唤上下文选项卡 -> 选择选项 -> 点击更改数据源，重新设置数据员范围，即可完成数据更新

2.2 新增行列

  • **新增行(计算项)**：对数据透视表内部的数据进行二次计算，形成新行；点击数据透视表行标签上，召唤上下文选项卡 -> 选择选项 -> 点击 域、项目和集 -> 点击计算项 -> 自定义新列名称和计算方式 -> 点击确认，即可完成新增行
  • **新增列(计算字段)**：对数据透视表内部的字段进行二次计算，形成新列；点击数据透视表，召唤上下文选项卡 -> 选择选项 -> 点击 域、项目和集 -> 点击计算字段 -> 自定义新列名称和计算方式 -> 点击确认，即可完成新增列

2.3 筛选

  • 切片器：对数据透视表进行筛选；点击数据透视表，召唤上下文选项卡 -> 选择选项 -> 点击 插入切片器 -> 选择合适的筛选列，可插入多个切片器
  • 筛选：点击数据透视表，召唤上下文选项卡 -> 选择选项 -> 点击 字段列表(最右侧) -> 显示数据透视表字段列表 -> 将筛选字段拉入筛选框 -> 点击出现的筛选框，可按照取值进行筛选（可筛选多项）
    拆分表：完成筛选后，点击数据透视表，召唤上下文选项卡 -> 选择选项 -> 点击选项下拉框(最左侧) -> 点击 显示报表筛选页，可以实现按照筛选值创建新的 Sheet

2.4 合并工作表

数据透视表和数据透视图向导：（需要格式相同、内容相似）

  1. 默认在功能区里面没有，需要 新增 ：点击文件菜单栏 -> 点击选项 -> 选择自定义工功能区 -> 在左侧筛选框筛选 不在功能区的命令 -> 将 数据透视表和数据透视图向导 添加至合适的功能选项卡（点击新建组，通常为数据组下 -> 选中新建组，选择重命名 -> 点击添加 -> 将 数据透视表和数据透视图向导 添加至新建的数据组中 -> 点击确认，完成添加功能区）
  2. 打开 数据透视表和数据透视图向导（快捷键：Alt + D + P）-> 选择 多重合并计算数据区域，点击下一步 -> 默认选择 新建单页字段 -> 选择 需要合并计算的区域，点击添加，完成全部添加后，点击下一步 -> 选择 数据透视表显示位置为 新工作表，点击完成

2.5 其他

  • 排序：点击行标签下拉框 -> 选择其它排序选项 -> 选择排序方式(升序和降序) -> 选择排序列标签 -> 点击确定，完成排序
  • 表格样式调整：
    1. 关闭加减按钮：点击数据透视表，召唤上下文选项卡 -> 选择选项 -> 点击 +/- 按钮(最右侧)，实现 +/- 按钮 的开启关闭
    2. 表格形式显示：点击数据透视表，召唤上下文选项卡 -> 选择设计 -> 点击报表布局下拉框 -> 选择 已表格形式显示
    3. 合并居中：点击数据透视表，召唤上下文选项卡 -> 选择选项 -> 点击选项下拉框(最左侧) -> 在布局和格式中 勾选合并且居中排列带标签的单元格
  • 开关字段对话框：点击数据透视表，召唤上下文选项卡 -> 选择选项 -> 点击 字段列表(最右侧)
  • 图形展示：点击数据透视表，召唤上下文选项卡 -> 选择选项 -> 选择数据透视图 -> 选择合适图形，即完成对数据透视表的图形化展示（可以进行数据切换）

数据分析：用于计算两组数据的相关关系，默认在功能区里面没有，需要新增

  • 新增：点击文件菜单栏 -> 点击选项 -> 选择加载项 -> 点击转到(最下方) -> 勾选分析工具库 -> 点击确定，完成 数据分析 功能的添加，默认在数据选项卡中
  • 使用：先选中需要进行计算的数据区域(纯数据区域) -> 点击 数据菜单栏 -> 点击 数据分析 -> 选择相关系数，点击确定 -> 选择输出数据区域 -> 点击确定，完成相关系数的计算
